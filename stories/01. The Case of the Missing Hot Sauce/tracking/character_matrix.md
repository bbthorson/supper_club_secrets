# Character Matrix — Book 1: The Case of the Missing Hot Sauce

A chapter × character snapshot. One row per chapter. Each character cell carries register and a one-line emotional state.

## How to read the cells

**Register** is one of: `public`, `private`, `under-pressure`, or a transition (`private → under-pressure`). Matches the three-register framework in `world building/02_character_voice_guide.md`.

**State** is a one-line emotional descriptor.

**Empty cell** = character is not in the scene (off-page or unmentioned).

**Cell with register but no state line** = character is present but reduced to set dressing. This is a flag for the author.

**⚠** prefix in a cell = the audit surfaced an issue for that character in that chapter. See `subplot_threads.md` for action.
